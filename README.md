# Engineering Graphics Using Scilab

Interactive Engineering Graphics, AI Simulations, and Educational Visualizations using Scilab.

---

# Project Overview

This project provides animated engineering graphics constructions and educational simulations using Scilab.

The software includes:

- Parabola using eccentricity method
- Ellipse using eccentricity method
- Hyperbola using eccentricity method
- Cycloid generation
- Involute of circle
- Animation-based geometry visualization
- Engineering education tools
- YouTube Shorts compatible simulations

---

# Features

## Engineering Graphics Modules

- Parabola construction
- Ellipse construction
- Hyperbola construction
- Cycloid curve
- Involute curve

---

## Animation Features

- Step-by-step curve generation
- Moving point animation
- Focus and directrix visualization
- Distance measurement display
- PF / PD eccentricity verification
